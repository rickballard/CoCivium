  `# 🟢➕ DO <Letter> — YYMMDD-HHMMZ — Short title`

- End-of-instruction demarcation: final line is a **timestamp-only** code fence:

- Colors: 🟢 safe/observe, 🟡 relax/toggle, 🟦 enforce, 🟥 risky/destructive, 🟫 optional.
