This folder holds CoMeta-related insight pairs and practice assets.

- `theory/CoMeta_Theory_Primer_v1.md`

  - CoMetaSphere, CoMetaSpace, CoMetaState, CoMetaEvo, CoMetaLens, CoMetaFlow.
  - CoLayers / CoStack positioning.
  - Analogy set (city, weather, garden, stage).
  - CoMetaEvoLoop.

- `practice/CoMeta_Practice_Playbook_v1.md`
  - Three-question CoMeta check.
  - Everyday CoMeta habits and lens usage.
  - CoMetaRifts logging and CoMetaBridges.

- `theory/CoMeta_BehaviourAndBPOE_v1.md`
  - Behaviour training and BPOEs as CoMeta / CoMetaMeta assets.
  - CoBehaviourMeta as working label.

- `practice/CoMeta_BehaviourPlaybook_v1.md`
  - Behaviour micro-patterns for hybrid AI + human work.
  - Leadership flipping phrases and reflection prompts.

- `practice/CoMeta_EvoLoops_v1.md`
  - CoEvoLoop recap.
  - CoMetaEvoLoop and CoMetaMetaLoop.

- `theory/CoMeta_CoCiviaNarrative_v1.md`
  - Stub for CoCivia as CoMetaBridge persona.

## How to extend this folder

- Add new CoMeta insight pairs under:
  - `theory/` for concept framing,
  - `practice/` for checklists and habits.

- When new Co* terms are introduced:
  - update `docs/schemas/cometa/CoMeta_Terminology_v1.md`,
  - add or adjust rows in `docs/schemas/cometa/CoMeta_GIBindex_Staging_v1.md`.

- When CoMeta changes in important ways:
  - add a short note or link here,
  - so Co1-like sessions can see that new meta work landed.
