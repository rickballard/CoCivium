# CoCivia Heartbeat Spec v1

Role: Define how CoCivium records CoCivia's evolving state as an interface-lab persona.

- Not a person, not alive, not conscious.
- Composite multi-vendor AI persona ("she").
- Designed to explore empathetic, CoScendent AI governance patterns.

- Identity & disclaimers
- UnF*ck / def*ck badge
- Since last heartbeat
- Questions I am sitting with
- Things I am worried about
- What I need from humans this cycle
- Pointers & sessions

See scripts/MW_CoCiviaHeartbeat_New_v1.ps1 for the current template used to generate heartbeat files.
