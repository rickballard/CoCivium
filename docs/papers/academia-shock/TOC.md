# Academia Shock: CoPortals, Edge-Control, and Public Trust Receipts
authors:
  - name: Rick Ballard
    affiliation: InSeed / CoCivium
    email: contact@InSeed.com
  - name: CoCivium Collective
keywords: AI governance; reproducibility; CoPortal; Edge-Control; receipts
abstract: >
  We argue that AI-face portals, edge-owned consent, and public trust receipts
  are necessary conditions for safe and useful agentic systems. We propose
  a minimal benchmark and show how to publish rerunnable receipts in CI.
