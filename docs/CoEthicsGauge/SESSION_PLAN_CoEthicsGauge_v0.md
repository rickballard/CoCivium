  - human (or circle) sets scores and signs off.

### Wave 3 — Lenses & Log Format

  - numeric core + Spanks/Nods band,
  - evidence + contributor metadata.
- Add 1–3 demo entries as worked examples.

### Wave 4 — v1 Launch Prep

- "Gauge your first brand in 60 minutes" how-to.
