# Filename Standards (CoCivium repos)

Use lowercase-kebab for files and folders. Reserve uppercase for acronyms.

- Primary content: `faiths-in-dialogue.md`
- Sidecar mapping (CoRef): `faiths-in-dialogue.coref.json`
- Concept registry (per repo or shared): `concept_registry.coref.json`
- Document metadata: `faiths-in-dialogue.meta.json`
- Images: `assets/faiths-mandala.svg`, `assets/faiths-mandala.png`

Avoid versioning in filenames; keep versions inside metadata and sidecars.
